About MehendiHub

A passion project built with love for Eid celebrations.

What is MehendiHub?

MehendiHub is a free Mehendi design gallery created to help people find the perfect henna design for Eid. Browse hundreds of Simple, Arabic, Bridal, and Kids designs — all in one place.

Every design can be saved to your favorites and downloaded for free. No account needed.

Built by Mozahid

This project was built as part of a web development portfolio. It showcases skills in HTML, CSS, and JavaScript including DOM manipulation, localStorage, responsive design, and clean UI architecture.
